# NightBite – Food Rescue Marketplace

Save food, save money, save the planet. Get mystery boxes from top restaurants at 50-70% off!

## Project Setup

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>

# Step 2: Install dependencies
npm install
```

### Development

```sh
# Start the development server with auto-reloading
npm run dev
```

### Building for Production

```sh
# Build the application
npm run build

# Preview the production build locally
npm run preview
```

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

To deploy this project, you can use popular hosting providers like Vercel, Netlify, or Cloudflare Pages.

For example, to deploy on Vercel:
1. Install the Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the project root to deploy.
